### 1、插入排序
> 从无序数组第二个数据项开始遍历剩下数组元素，

> 如果该项小于于上一个元素，遍历前部所有大于该项的元素，前边元素后移一位，最后一个遍历位置，就是该项的位置

```
    function insertSort($arr)
    {
        // 从无序数组第二个数据项开始遍历
        for ($i = 1; $i < count($arr); $i++) {
            // 该项小于于上一个元素
            if ($arr[$i] < $arr[$i - 1]) {
                $tmp = $arr[$i];
                // 遍历前部所有大于该项的元素
                for ($j = $i - 1; $j >= 0 && $arr[$j] > $tmp; $j--) {
                    // 前边元素后移一位
                    $arr[$j + 1] = $arr[$j];
                }
                // 最后一个遍历位置，就是该项的位置
                $arr[$j + 1] = $tmp;
            }
        } 
        return $arr;
    }
    $arr = [9, 1, 15, 13, 4, 19, 2];
    print_r(insertSort($arr));
```
