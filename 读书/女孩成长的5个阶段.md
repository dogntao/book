### 女孩成长的5个阶段
#### 1、从出生到2岁
> 安全感,我是否被爱,是否安全
```
    婴儿不会自己安抚自己,所以容易烦躁,哼唧,嚎啕大哭
    照顾者需要帮她们平静下来,抱起来,安慰她,帮她放松
    宝宝需要平静和安抚,同时需要伴随兴奋,快乐.跟爸爸一起疯玩长大的孩子比玩温和游戏的孩子,抗压能力高的多
    陪孩子玩是非常重要的事,玩耍能激发大脑的潜力,爱玩的任善于创新.开心环境长大的孩子不畏缩,不木讷,活力十足
    不建议早教,不要把孩子交给陌生人,她们真正需要的是你
```
#### 2、从2岁到5岁
> 探索,世界是否有趣快乐
```
    艺术、运动能力,是从1~5岁阶段开始的
    孩子玩具堆积成山,她们无从下手,给她们准备几样玩具,放到方便易取的盒子里,经常换几样就可以了
    孩子需要自己的空间,需要白日梦和无所事事的空间,这样想象力才能得以发展
    要温和但坚定地教她们纪律和规矩,帮助她们处理自己的感受
```
#### 3、从5岁到10岁
> 与人交往,跟人是否能好好相处
```
    小婴儿不会对一样的婴儿有太大兴趣,3岁以下的孩子在一起能好好相处的时间超不过几分钟
    4-5岁可一起玩幻想和假扮的游戏,这些游戏是培养孩子创造力的基础,同时玩耍的过程中学会了如何与他人沟通
    同龄男孩干一架,可能一小时后就又在一起厮混了,但女孩没那么容易走出来,需要家长们关心她,抚慰她,但大部分不用干预.
    在她求助时,帮她分析,给与建议
    交友的7个建议:友好、分享、同理心、控制攻击性、道歉、理解情绪和什么人可以信任
```
#### 4、从10岁到14岁
> 了解内心,发现自我,发现真正的快乐
```
    这个阶段女孩需要更多的关注和陪伴,需要多教导、多解释、多询问,参加更有意义的活动,不是把她们推出去独立自成长
    教育不是装满一桶水,而是点燃一把火
    青少年都具有某种火苗,三大类:天赋(写作、绘画),性格(勇气、同理心),承担某项任务(保护大自然),她们能从中得到巨大的快乐,
    父母的任务就是发现和强化孩子的火花,这些火花能让孩子在学校表现更快乐、自信、健康
    发现灵魂,寻找榜样
```
#### 5、从14岁到18岁
> 踏入成人世界,对自己负责任
```
    14岁大脑还没有完全发育成熟,有压力、紧张时会冲动,她们仍然需要父母而不是朋友
    规定晚上几点必须在家,禁止饮酒,随时知道她在哪,和谁在一起
    学会自立、思路清晰、情感强大、处理好和别的关系、做事有目的性
    女孩自己要为自己负责、保持清醒和察觉

    过早的性感不要,区分喜欢、爱和色欲
    喜欢异性正常,恋爱会有什么样的问题,色欲坚决戒除
    管好孩子接触信息的渠道,孩子卧室不能放电视,限制孩子上网、电子产品和社交产品的使用或使用时间,并在青少年适宜的环境下使用
    妈妈要担负起女儿在性教育问题上的教育

    阻止坏女孩
    朋友被欺凌时,要勇敢的站出来反对,这样那些坏女孩也不敢欺凌她,即便发生了她的朋友也会帮她,如果没有家长要有所作为,采用平静、友好而坚决的态度与校方、老师、对方的家长沟通,跟进处理效果,如果无效,坚持转学,爸爸在这类事中要站出来,为女儿打气,做女儿的靠山

    和孩子进行深度谈话,让她明白网络的便捷,也要知道网络上到处存在的陷阱和阴暗面
    可以帮孩子注册社交账号,设置最大承德的隐私保护,告诉海西父母只是关心你的安全
    和孩子保持畅通沟通渠道,她看到什么不开心的事都会跟你说,你也要保持足够的耐心和警觉

    妈妈是女儿的榜样,妈妈让她们懂得了成为一个女人的意义.在她们不同的人生阶段,她们会崇拜妈妈、憎恨她、敬仰她、指责她,
    但没有女儿会认为妈妈不重要

    爸爸如果在培育女儿上投入越多,女儿就会表现越自信、成绩更好
    爸爸们做更多的家务,女儿就会更有自信选择有挑战的工作,如领导人、科学家

    爸爸和妈妈对女孩的作用是互补的,妈妈让女孩安心,爸爸让女孩感到自信,而且爸爸与女儿的关系是她以后与男性相处的基础
    爸爸尊重她,听她的意见,她就不会接受别的男孩不尊重她
    爸爸经常和她开玩笑,和她讨论一些深刻的话题,她会更加自信,不会被别操纵

    青春期的女儿情绪是很难控制的,她们的大脑要么反击,要么逃亡,很难听得进别人的意见,喜欢迷恋一些疯狂的事情,但那都是短暂的.
    想要帮助她,倾听、耐心、温和、幽默、同理心而不是道理、指责
```
